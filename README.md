## Installation
Pick either Node or Docker way
### 1a. Node
- `npm i`
- `npm run dev`

### 1b. Docker
- `docker compose up -d`

### N8N
- Import workflow `N8N - LHKPN.json`
- Open ENV node to update the host
- Add Telegram Bot Credentials
- Activate the workflow
